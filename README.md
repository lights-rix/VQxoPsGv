# 项目简介

## 核心功能
- 基于YOLOv8的道路交通信号检测系统，已训练好模型
- 支持图片、视频及摄像头进行实时检测
- 支持选择不同模型进行检测
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图

## 适用场景
- 道路交通信号检测与分析
- 智能交通监控系统
- 交通标志识别研究
- 自动驾驶辅助系统开发
- 智能监控与安防系统

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 包含2093张标注数据集，涵盖21类交通标志
- 提供完整源码、数据集、训练好的模型及运行教程
- 图形化界面，操作简单直观
